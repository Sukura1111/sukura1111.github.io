java基础

1：什么是面向过程和面向对象

面向过程

  • 特点：强调过程

  • 面向过程是什么

    • 当需要实现一个功能时，每个具体步骤都亲力亲为，详细处理每一个细节

面向对象

  • 特点：强调对象

  • 面向对象是什么

    • 当需要实现一个功能的时候，不关心具体的步骤，而是找到已经有该功能的人，来帮我做事

2 ：类和对象

定义类

  • 注意事项

    • 1：成员变量是直接定义在类当中的
    • 2：成员方法不能用static修饰

  • 例子

    • public class student{

      //成员变量

      String name；

      //成员方法

      public void eat(){

      System.out.println(“吃饭”)

      }

}

定义对象

  • 创建，格式

    • 类名称 对象名 = new 类名称（）

  • 使用

    • 1：对象名.成员变量名

      • student.name

    • 2：对象名.成员方法名

      • student.eat( )

3：局部变量和成员变量

1：定义的位置不一样

  • 局部变量

    • 在方法的内部

  • 成员变量

    • 在方法的外部，直接写在类当中

2：作用范围不一样

  • 局部变量

    • 只有方法当中才可以使用，出了方法就不能再用

  • 成员变量

    • 整个类全都可以通用

3：默认值不一样

  • 局部变量

    • 没有默认值，如果想要，必须手动进行赋值

  • 成员变量

    • 如果没有赋值，会有默认值，规则和数组一样

例子

  • public class demo1 {
    //成员变量
    private String name;

    public void eat(){

    //局部变量
int num = 20;

    }
    }

4:封装

封装在java中的体现

  • 1：方法就是一种封装

    • public static void eat(){
      System.out.println(“hello”);
      }

  • 2:关键字private也是一种封装

    • 一旦使用了private进行修饰，在本类中可以随意访问，但是超出本类就不能直接访问了

      • 间接访问private成员变量，就是定义一对Geeter/Setter

        • 对于Getter来说，不能有参数，返回值类型和成员变量对应
        • 对于Setter来说，不能有返回值，参数类型和成员变量对应

    • private封装的好处

      • 可以防止main在调用的时候随意赋值参数

什么是封装

  • 封装就是将一些细节信息隐藏起来，对于外界不可见

5：构造方法

1：什么是构造方法

  • 构造方法时专门用来创建对象的方法，当我们通过关键字new来创建对象的时，其实就是在调用构造方法

2：格式

  • pubic 类名称（参数类型 参数名称）{
    方法体
    }

3：注意事项

  • 1：构造方法的名称必须和所在的类名称完全一样，连大小写也要一样

  • 2：构造方法没有返回值，连void都不用写

  • 3：构造方法不能return一个具体的返回值

  • 4：如果没有编任何构造方法，那么变压器会默认赠送一个构造方法，没有参数，方法体什么都不用做，

  • 5：一旦编写了至少一个构造方法，那么编译器将不再赠送

  • 6：构造方法也是可以进行重载的

    • 重载：方法名称相同，参数列表不同

6：匿名对象

什么是匿名对象

  • 匿名对象就是只有右边的对象，没有左边的名字和赋值运算符。new 类名称（）；

注意事项

  • 匿名对象只能使用唯一的一次，下次再使用不得不再创建一个新对象

使用建议

  • 只有确定有一个对象只需要使用唯一的异常，就可以使用匿名对象
